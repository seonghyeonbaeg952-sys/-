## 점수 해석과 검증 범위

- 점수 순서는 `핵심 과업 탐색 / 정보 위계 / 브랜드 고유성 / 모바일 / 모션 유용성·제어 / 접근성 신호`이며 각 1–5점이다.
- 비교 총점은 `취득점수 ÷ 확인 축 만점 × 30`으로 정규화했다. 예: 세 축 합계 13점이면 `26.0/30 (3축)`이다.
- 실제 확인하지 않은 모바일, 키보드, 저모션 상태는 추정하지 않고 `N/A`로 남겼다. 접근성 축은 키보드 흐름까지 확인하지 않아 전부 `N/A`다. 이는 부적합 판정이 아니라 미확인이다.
- 390px 모바일은 연결 브라우저에서 성공적으로 다시 연 12개만 점수화했다. 단순 viewport meta 존재를 모바일 품질로 간주하지 않았다.
- HTTP 403·429·대기열·타임아웃은 사이트 부재가 아니다. 이 경우 기존 공식 페이지 관찰 또는 검색 색인으로 확인 가능한 범위만 기록하고, 현재 UI·모바일·동적 상태는 추정하지 않았다.
- `관찰 보류`는 확인 축이 4개 미만이라 상위군 후보에서 제외된다는 뜻이다. 정규화 점수가 높아도 확인 범위가 좁으면 상위군으로 올리지 않았다.
- 모바일 재검증의 작은 터치 대상 집계에는 본문 인라인 링크도 포함될 수 있어 보조 신호로만 사용했다. 가로 오버플로, 메뉴의 접근 가능한 이름, 고정 레이어를 함께 보았다.

## 9. SMYC에 적용할 결론

1. 홈은 `정체성 한 문장 → 다음 공연 → 듣기/교육 → 검증 가능한 후원 근거`의 순서가 가장 안정적이다.
2. 네이비·아이보리·오렌지/골드의 역할을 고정하고 색은 포스터·공연 이미지에서 추가한다.
3. 공연 카드 메타는 `날짜 → 제목 → 장소 → 대상/상태 → CTA` 순서를 모든 viewport에서 유지한다.
4. 청소년 공개 화면은 개인 초상보다 악보·손·공간·그룹 활동·익명 결과물을 우선한다.
5. 성인합창단의 1989년 출범·1990년 직업체제 전환과 청소년합창단의 2014년 출범을 섞지 않는다.
6. 카드 그림자·둥근 모서리보다 여백·규칙선·타입 크기·원본 이미지 비율로 위계를 만든다.
7. 모션은 상위군을 아직 확정하지 않는다. SMYC 구현에서 pause, 키보드, 저모션 대체, 상태 보존을 직접 검증한 뒤 채택한다.
8. 교육·입단 상위군도 모바일 확인이 부족하다. Interlochen·Berklee·Juilliard를 다음 브라우저 검증 우선순위로 둔다.
