## 1. 조사 방법과 판정 기준

- 단순히 “예쁜 웹사이트”를 모으지 않고, 디자인 시스템·접근성·모바일·성능·전환·문화예술/비영리 스토리텔링·웹 모션을 함께 조사했다.
- URL은 검색 결과와 원문 페이지를 교차 확인했다. YouTube는 개별 영상의 유행 수명이 짧아, 2025–2026에 계속 교육 자료를 발행하는 채널을 중심으로 선정했다.
- `공식`은 W3C, Apple, Google/Chrome, Figma, Webflow, Framer 등 해당 표준·도구·플랫폼의 발행처가 직접 제공한 자료라는 뜻이다. `독립`은 교육자·리서치 기관·매체·전문 회사의 자료다.
- `최신`은 2025–2026 발행 또는 2026년 현재 유지·갱신되는 문서, `기초`는 발행 연도보다 원칙의 지속성이 중요한 상시 문서다.

- 트렌드 채택은 “여러 출처에서 반복됨”, “접근성·성능에 위배되지 않음”, “합창단의 신뢰·격조·공동체 서사에 맞음”을 모두 만족할 때만 권장한다.

### A06. 생성형 AI풍 3D·그라디언트·완벽한 스톡 이미지

- 문제: 사이트를 최신처럼 보이게 할 수 있지만 합창단의 실제 정체성과 신뢰를 약화한다.
- 근거: R03, R38, R39, R41
- 대안: 실제 장소·공연·악보·손의 흔적과 지역 문화에서 시각 언어를 만든다.

### A07. 숨은 내비게이션과 아이콘만 있는 핵심 CTA

- 문제: 기능을 발견하기 어렵고 처음 방문자·고령 보호자·보조기술 사용자가 길을 잃는다.
- 근거: O11, R24, R25, R26
- 대안: 핵심 행동은 항상 텍스트 라벨을 보이고 보조 아이콘만 함께 사용한다.

### A08. 효과 쌓기: 저대비 골드 + 블러 + 그림자 + 텍스처 + 모션

- 문제: 개별 유행을 모두 섞으면 정보 계층·성능·브랜드 절제가 동시에 무너진다.
- 근거: O05, O24, R01, R05, Y09
- 대안: 화면마다 하나의 주인공 효과만 고르고 나머지는 타이포·여백·명확한 상태로 받친다.

## 7. 5개 Figma 샘플로 연결할 때의 권장 실험 축

이 자료의 목적은 다섯 개 모두를 서로 다른 장식 스타일로 만드는 것이 아니다. 공통 정보 구조와 접근성 기준은 고정하고, 아래처럼 하나의 명확한 가설만 달리해야 비교할 수 있다.

1. **Editorial Stage** — Navy 무대와 Ivory 지면을 오가는 편집적 구성, 공연 포스터 전환.
2. **Living Score** — 악보의 선과 호흡을 스크롤 진행으로 연결하되 reduced-motion에서는 정적 선으로 대체.
3. **Warm Archive** — 연혁·공동체 기록·실제 자료의 촉감, 카드→상세의 공유 요소 전환.
4. **Modern Sacred** — 교회음악의 절제·빛·공간을 현대적 그리드와 고대비 타이포로 표현.
5. **Next Generation** — 청소년의 생동감은 개인 사진 노출이 아니라 색 리듬·집단 활동·교육 과정으로 표현.

다섯 샘플은 모두 홈 화면만 보여주는 정지 목업이 아니라 `홈 → 공연 상세`, `홈 → 입단`, `홈 → 후원/문의`, `모바일 메뉴`, `reduced-motion` 중 최소 3개 흐름을 포함해야 한다.

## 8. 적용 전 검증 체크리스트

- 390px, 768px, 1440px에서 긴 한국어 제목·빈 데이터·느린 이미지까지 확인한다.
- 모든 핵심 타깃은 44px 이상, 텍스트 대비는 WCAG 2.2 AA 이상을 유지한다.
- 키보드만으로 메뉴·카드·모달·폼을 완료하고 포커스가 가려지거나 갇히지 않는지 확인한다.
- `prefers-reduced-motion: reduce`에서 이동·확대·패럴랙스·자동 루프를 제거한다.
- LCP, INP, CLS 예산을 넘기는 영웅 영상·블러·3D·JS 모션은 삭제하거나 지연한다.
- 실제 CMS 데이터, `is_visible = true` 조건, 빈 상태, 로딩·오류 상태를 시안에 반영한다.
- 청소년 단원의 개인정보와 초상 노출을 최소화하고 공동체·활동·파트·그룹 중심으로 표현한다.
- 시안 평가는 감성(브랜드), 이해(정보 구조), 행동(전환), 포용(접근성), 현실성(구현·운영) 다섯 축으로 진행한다.
