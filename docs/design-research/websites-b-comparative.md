- 100개를 같은 여섯 축으로 평가했다: 핵심 과업 탐색, 정보 위계, 브랜드 고유성, 모바일, 모션 유용성·제어, 접근성 신호.

## SMYC 구성 결론

1. **감성층과 과업층을 겹치지 않는다.** 히어로는 실제 무대 자산과 한 문장으로 정체성을 전달하고, 그 위 또는 바로 아래에 다음 공연 날짜·장소·상태·CTA를 고정한다.
2. **공연·교육·모집을 하나의 이벤트 모델로 묶는다.** 텍스트 상태 태그와 공통 날짜 필터를 쓰되 청소년·무료·마감·접근성 정보를 색만으로 구분하지 않는다.
3. **아카이브는 포스터 갤러리가 아니라 근거 있는 기록이다.** 날짜, 장소, 프로그램, 역할, 출처, 공개 동의를 메타데이터로 보존한다.
4. **입단은 단계형 서비스다.** 대상, 접수 기간, 준비물, 오디션, 결과, 첫 연습을 한 흐름으로 제공하고 종료 공고는 자동으로 보관한다.
5. **모션은 기존 애니메이션을 재안무한다.** 인트로를 짧게 하고, 오선·호흡·성부 합류 모션은 상태와 관계 설명에만 배치한다.
6. **미성년자 보호가 비주얼 방향을 결정한다.** 개인 얼굴 카탈로그보다 전체 무대, 악보, 손, 공간, 조명, 연습 과정의 비식별 장면을 우선한다.

## 근거·한계

- 원 URL과 1차 관찰은 `docs/design-research/websites-b.md`에서 가져왔고, 점수는 `docs/design-research/site-comparison-rubric.md`를 적용했다.
- 평가는 로그인 전 공개 화면과 조사 당시 시즌에 한정된다. 지역·쿠키·캠페인 변경으로 화면이 달라질 수 있다.
- 모바일, 키보드, 스크린리더, 색 대비 수치, 저모션 미디어쿼리는 별도 실기 감사가 아니므로 확인되지 않은 축은 `N/A`다.
- 정규화는 미확인 축에 점수를 부여하지 않기 위한 비교값이다. 2~3축의 높은 값은 완성도 우위가 아니며 상위군에서 제외했다.
- `X` 3개 사이트는 자동 수집 차단 상태다. 직접 브라우저 캡처 전 어떤 현행 UI 주장도 설계 근거로 사용하지 않는다.
