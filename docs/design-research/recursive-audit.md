### 감사 방법

- Markdown 표는 ID, 열 수, 행 수, 빈 셀, URL을 PowerShell 정규식과 URI 파서로 재계산했다.
- URL canonicalization은 host 소문자화, `www.` 제거, 루트 이외 끝 `/` 제거를 사용했다.
- A/B 원목록과 비교표의 `번호 ↔ ID ↔ URL`을 행 단위로 대조했다.
- 점수는 숫자가 있는 축만 합산하고, 문서에 적힌 확인 축 수·30점 환산값과 재계산값을 비교했다.

- 외부 링크는 병렬 `curl -I -L`로 확인하고, timeout·`HEAD` 거부·404 후보는 개별 `GET` 또는 더 긴 timeout으로 재확인했다.
- `독립 출처 수`는 URL 수가 아니라 발행 주체·연구 데이터·작성자 계열이 다른 provenance family 수로 계산했다.
